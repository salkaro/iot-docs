## How to Generate and Copy an API Key

To use the API, you need to generate an API key. Follow these steps:

1. Go to [https://iotapp.salkaro.com/settings#authentication](https://iotapp.salkaro.com/settings#authentication).
2. Click on **Generate New**.
3. Give your API key a descriptive name and set the appropriate permissions.
4. Click **Create API Key**.

> **Important:** Keep your API key secure and do not share it publicly.
